

# CI/CD Projekt - Vue + GitHub Actions

## Projektbeskrivning
Detta projekt är en enkel Vue-applikation byggd med Vite och TypeScript.  
Syftet med projektet är att demonstrera en CI/CD-pipeline med GitHub Actions.

## Vad jag har gjort
- Skapat Vue + TypeScript projekt med Vite
- Skrivit en enkel funktion och tester med Vitest
- Konfigurerat CI/CD pipeline med GitHub Actions
- Automatisk build, test och deployment
- Deploy till GitHub Pages
